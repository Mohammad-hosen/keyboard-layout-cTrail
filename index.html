Bangla Keyboard Layout is an application program forAndroid that adds bangla keyboard layout to the system keyboard layouts, which works when external physical keyboard* is used.

Purpose of the app

It may sound odd, who needs to type Bangla on external physical keyboard on their phone or tab instead of using virtual (software) keyboard. Typing on virtual keyboard just works fine for all of us. But typing on physical keyboard is way more easy, fast and convenient. If you know how to type Bangla on the physical (desktop) keyboard and you have one or buy one but any computer except your Android device in not available to you now, then it is a useful app to you.

In most cases we learn to type bangla on the Bijoy keyboard layout**. Also almost all bangla desktop keyboards use Bijoy layout to map their keys. But on Microsoft Windows or Android OS, preloaded system layout for Bangla language is totally different. That's why when you try to type something in bangla using physical keyboard the letters on the screen and that on the keyboard keys don't match. And so we have to use the Bijoy Classic, Bijoy Bayanno, Avroetc app on our windows computers to match the letters of the keyboard with the resulting input. These apps on windows works as adds-on to the Microsoft office programs. But we don't find such program for android devices.

As a necessity I had to make this app for my won use during a training on typing. I think it may help some others too. It works well on android device and has little difference between it and the Bijoy layout.

How does it work?

The app doesn't have any interface, it just adds the layout to the system preloaded layouts. You won't be able to open or find the app in the app drawers. It only shows up in the Settings/Apps menu. If you use any virtual (software) keyboard then it will use its own layout. Layout of this app can be used for physical keyboard only. After connecting a physical keyboard, keyboard layout selection option will be available under the settings/languages_and_input /physical_keyboard menu (or in the notification panel if you are connecting the keyboard for the first time after a boot.) You need to select this app's layout named BanglaBangla Keyboard Layout from the list. And you are good to go.

Why it is different from Bijoy layout?

Bijoy Classic, Bijoy Bayanno, Avro etc apps work as input modifier to the system and also work as adds-on to the office programs, can change the manipulation of letters and symbols. In one sence, these apps bypass the system manipulation of input to the office programs. This type of tasks needs special integration with other applications which is not supported on Android. So the app can't perform these previously mentioned operations. It just adds its layout to the system. When its layout is selected system use it and other input rules (which are controlled by system) remain unchanged. As a result there are few things, you can complain about:

  1. There is no dedicated key for ref, ্র, ্য ; since they are not separately defined in unicode. Infact they are composite of multiple(three) unicode and Android Key Character Map does not support multiple unicode on a single key.

  2. You have to type the letter first and then vowel mark (“কার” যেমন: া, ে,ি). It is the input method of Android OS.

  3. For the vowel letters (except অ), input method is different form bijoy. In bijoy you need to type [ ্] + [vowel marks ( া, ে, ু, ী)] and the apps such as Bijoy Bayanno, Avro etc. (on Windows) replace them with the corresponding vowel letters. Here you need to Press the [Alt] + [the vowel mark] or [Alt] + [Shift] + [vowel mark] at a time to input the Corresponding vowel letter.

  4. The key [Z] contains ‘ৎ’ and ‘ঃ’; [A] key contains ‘’ and ‘ ৄ ’ to utilize the keys.

  5. There are also few changes in the vowel mark keys to make typing easy.

How to use the app

Follow the steps below:

  1. Install the downloaded app.
  2. Add Bangla as additional language to you current virtual (software) keyboard.
  3. Connect an physical keyboard to the Android device.
  4. If you are connecting the keyboard for the first time after a boot then you will see keyboard layout selection option in the notification panel. Otherwise go to Settings/ Languages & input/ Physical keyboard menu.
  5. Tap on the Bangla language option and a list of layouts will show up.
  6. Among these layouts select the one labelled as Bangla Keyboard Layout.
  7. Exit from setting. Open the app you want to type on. And you are ready to type Bangla using your physical keyboard.

If you want to switch to another language just Press the [Ctrl] + [Space] keys or [Windows/OS] + [Space] keys together. Do it again if you want to switch back.
